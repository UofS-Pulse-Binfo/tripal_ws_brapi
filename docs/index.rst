Tripal Web Service - BrAPI
##########################

.. image:: tripal-ws-brapi-module.png

A Drupal module that implements the Breeding API (BrAPI) standardized specifications
for Tripal websites that host genotypic/phenotypic data stored in a CHADO Database.
BrAPI is an interface for data interchange between crop breeding applications.
